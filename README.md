# Frontend - Aplicación de Gestión de Mascotas

## Descripción

Frontend de una aplicación Full-Stack para la gestión de mascotas, desarrollado como proyecto final del Máster de Programación Web.

La aplicación permite interactuar con una API propia para gestionar la información de las mascotas mediante una interfaz moderna y responsive.

---

## Tecnologías utilizadas

- React
- JavaScript
- HTML5
- CSS3
- Vite

---

## Funcionalidades

- Interfaz responsive.
- Consumo de API REST.
- Gestión de mascotas.
- Navegación entre páginas.
- Comunicación con el backend.
